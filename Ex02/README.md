## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：四則演算をする
- クリア：entryに入力されている数字，数式を全て消去する
- 小数点・()：小数点や()を式に追加する
- √:entryに入力されている数字，数式の平方根を計算して表示する
- 2乗：entryに入力されている数字，数式を2乗して表示する
- %：entryに入力されている数字，数式を%表示に変える
